Project Description

The goal of Brain Network is to advance communication and collaboration among peers. This is done by removing the barrier of a physical meeting. Brain Network aims to one day offer an experience that has all the qualities of an in person meeting. Brain Network will have features including:

  • A virtual whiteboard
  • Realtime video streaming
  • Text, voice, and possibly video chat
  • Screen Sharing

Use Cases

  • School or Work project brainstorming and collaboration
  • Solving problems with others. Math, logic, etc.
  • Watching educational or instructional videos in a group
  • Sharing work or information found on one device with a group in real time
  • Much More!
